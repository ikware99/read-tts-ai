# 中文小说语音阅读器

一个专为中文小说阅读设计的语音合成应用，基于浏览器原生 Speech Synthesis API，提供高质量的女性角色配音和智能阅读体验。

## 📋 项目简介

中文小说语音阅读器是一个基于 Web 技术开发的文本转语音应用，专为中文小说阅读场景优化。它结合了智能中文分词、专业女性角色配音和完整的播放控制功能，为用户提供沉浸式的语音阅读体验。

## ✨ 核心功能

### 1. 智能中文分词系统
- **智能分段**：基于标点符号和语义的自动分段，适合小说阅读
- **中文分词**：识别中文词语边界，为语音合成提供准确断句依据
- **分词可视化**：直观展示分词结果，支持段落点击播放

### 2. 专业女性角色配音
- **三种特色女声**：
  - 晓晓：年轻活力女声，适合现代小说
  - 晓伊：甜美温柔女声，适合言情小说
  - 云希：清晰标准女声，适合经典文学
- **语音参数调节**：
  - 语速：0.5x - 2.0x 可调
  - 音调：0.5x - 2.0x 可调
  - 音量：0% - 100% 可调
- **沉浸式体验**：专为小说阅读设计的语音风格

### 3. 完整的播放控制
- **播放控制**：播放/暂停/停止功能
- **段落导航**：支持点击任意段落直接播放
- **实时波形显示**：直观的音频波形可视化
- **进度跟踪**：实时显示播放进度和时间

### 4. 智能滚动与高亮
- **自动滚动**：播放时自动跟随阅读位置
- **智能高亮**：当前播放段落高亮显示
- **可切换功能**：连续两次双击空白区域可开关智能滚动和高亮
- **用户自由**：暂停时不自动滚动，允许自由浏览

### 5. 浮动控制栏
- **便捷控制**：页面右下角浮动控制按钮
- **智能折叠**：无操作5秒后自动折叠，只显示播放按钮
- **悬停展开**：鼠标悬停时自动展开所有按钮
- **完整功能**：包含播放/暂停、音量调节功能

### 6. 用户体验优化
- **响应式设计**：完美适配桌面和移动设备
- **现代化 UI**：温馨的粉色主题，符合女性配音的温柔特质
- **文件上传**：支持导入 TXT/MD 格式小说文件
- **示例文本**：内置《红楼梦》经典片段，快速体验

## 🛠️ 技术栈

- **前端框架**：纯 HTML5 + CSS3 + JavaScript
- **语音合成**：Web Speech Synthesis API
- **中文分词**：基于规则的智能分词算法
- **可视化**：HTML5 Canvas（波形显示）
- **样式设计**：Tailwind CSS + CSS3 Flexbox + Grid 布局
- **响应式设计**：Media Queries
- **状态管理**：函数式编程 + 全局状态对象

## 📖 使用方法

### 基本使用

1. **输入文本**：
   - 在文本输入框中粘贴小说内容
   - 或点击"上传文件"导入 TXT/MD 文件
   - 或点击"示例文本"加载《红楼梦》片段

2. **处理文本**：点击"处理文本"按钮进行智能分段和分词

3. **选择配音**：在"语音设置"中选择喜欢的女声

4. **调整参数**：根据需要调整语速、音调和音量

5. **开始阅读**：
   - 点击"播放"按钮开始播放
   - 或直接点击任意段落开始播放
   - 使用浮动控制栏进行便捷操作

### 高级功能

- **智能滚动控制**：连续两次双击空白区域可开关智能滚动和高亮
- **段落导航**：点击任意段落直接跳转到该位置播放
- **音量快捷调节**：使用浮动控制栏的音量按钮快速调节

## 🔍 功能详细描述

### 1. 智能中文分词系统

**实现原理**：
- 首先按换行符进行初步分段
- 然后基于标点符号（。！？；…）进行句子切分
- 使用规则引擎识别中文词语边界
- 针对常见词语和短语进行优化

**分词特点**：
- 适合中文小说的长文本处理
- 考虑了中文语法和语义结构
- 生成适合语音合成的自然断句

### 2. 语音合成与播放控制

**实现原理**：
- 基于浏览器原生 Web Speech Synthesis API
- 动态获取可用语音列表
- 支持多种语音参数调节
- 实现完整的播放状态管理

**播放控制**：
- 支持播放、暂停、停止操作
- 自动处理段落间的播放过渡
- 实时更新播放状态和进度

### 3. 智能滚动与高亮

**实现原理**：
- 监听播放状态和当前段落
- 使用 Intersection Observer API 检测元素可见性
- 结合 CSS 过渡实现平滑滚动
- 使用状态管理控制功能开关

**智能特性**：
- 播放时自动滚动到当前段落
- 暂停时停止滚动，允许自由浏览
- 连续两次双击可切换功能状态
- 高亮显示当前阅读位置

### 4. 浮动控制栏

**实现原理**：
- 使用 CSS 固定定位实现浮动效果
- 通过 JavaScript 控制折叠/展开状态
- 实现 5 秒无操作自动折叠
- 监听鼠标事件实现悬停展开

**设计特点**：
- 不遮挡主要内容
- 提供便捷的播放控制
- 智能适应用户操作
- 与整体 UI 风格统一

## ❓ 常见问题

### 1. 语音合成失败怎么办？

**解决方案**：
- 刷新页面重试
- 更换浏览器（推荐 Chrome 或 Edge）
- 检查浏览器语音设置

**原因**：
- 不同浏览器支持的语音列表不同
- 某些语音可能需要额外安装

**解决方案**：
- 尝试使用其他语音
- 更新浏览器到最新版本
- 检查浏览器语音设置

**方法**：
- 连续两次双击页面空白区域
- 在弹出的提示中确认关闭
- 再次连续双击可重新开启

### 4. 支持哪些文件格式？

- 支持 TXT 格式文件
- 支持 MD（Markdown）格式文件
- 支持直接粘贴文本内容

## 💡 技术说明

### 语音合成原理

本项目基于浏览器原生的 Web Speech Synthesis API，该 API 提供了文本到语音的转换功能。主要原理如下：

1. 创建 `SpeechSynthesisUtterance` 对象，设置要合成的文本和参数
2. 调用 `speechSynthesis.speak()` 方法开始合成和播放
3. 监听 `onend`、`onerror` 等事件处理播放状态
4. 支持多种语音参数调节，如语速、音调、音量等

### 中文分词算法

项目实现了一个基于规则的中文分词算法，主要特点：

- 基于常见词语和短语库
- 考虑中文语法结构
- 针对小说文本优化
- 支持长文本高效处理

### 智能滚动实现

智能滚动功能通过以下技术实现：

- 监听播放状态和当前段落
- 使用 `Element.scrollIntoView()` 实现平滑滚动
- 结合 `setTimeout` 实现延迟恢复
- 使用状态管理控制功能开关

## 📊 性能优化

- 优化分词算法，提高处理速度
